# peterjmags.com

[http://peterjmags.com/](http://peterjmags.com/)

This is my personal portfolio site, built primarily back in October 2012. A year and a half later, I'm open-sourcing it, partly to encourage myself to make some updates and improvements that have been nagging at me for a while, and partly because it's the cool thing to do.

### Built with real state-of-the-art 2012 stuff

- [Bootstrap 2.1.1](https://github.com/twbs/bootstrap/tree/v2.1.1)
- [staticjinja](https://github.com/Ceasar/staticjinja)
- [jQuery 1.8.2](https://github.com/jquery/jquery/tree/1.8.2)
- [Modernizr 2.6.2](https://github.com/Modernizr/Modernizr/tree/v2.6.2)
- [Swipe 1.0](https://github.com/bradbirdsall/Swipe/tree/1.0)
- [Respond.js 1.1.0](https://github.com/scottjehl/Respond/tree/1.1.0)
